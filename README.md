# SPF Me Up

SPF Me Up is a simple web application that helps users determine whether they
should apply sunscreen based on the UV index at their current or selected location.

## Features
- Fetches real-time UV index using a public weather/UV API
- Supports both current location and custom location input
- Provides simple recommendations on SPF usage

## Tech Stack
- JavaScript
- Node.js
- Express
- EJS
- External UV Index API

## Purpose
This project was built to practice API integration, backend logic, and basic
decision-based recommendations.
